>[!sidenote]- **OBSERVACIÓN** 
>Para los puntos cardinales laterales, el ángulo es 45°, es decir:
>- NE = N(45°)E
>- SE = S(45°)E
>- NO = N(45°)O
>- SO = S(45°)O
